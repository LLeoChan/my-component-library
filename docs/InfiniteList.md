# InfiniteList

## Description

The `InfiniteList` component provides an infinite scrolling list that loads more items as you scroll.

## Usage

```vue
<template>
  <InfiniteList />
</template>

<script>
import InfiniteList from 'my-component-library/components/InfiniteList.vue';

export default {
  components: {
    InfiniteList,
  },
};
</script>
